# BBVA REST-API

## Getting Started

An API created for BBVA Hackathon challenge, created with *Node* and **ExpressJS**.

# Dependencies

[**express**](https://www.npmjs.com/package/express)
```sh
yarn add express
```

## Dev-Dependencies

[**nodemon**](https://www.npmjs.com/package/nodemon)
```sh
yarn add nodemon --dev
```
